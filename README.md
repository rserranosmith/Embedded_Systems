### `bit_twiddling.c`

This program is an introduction to bitwise operations. It turns on and off ports to created a sweeping motion with LED's. A button is used to reset and turn off the LED's.

#### Parts List

- 4 LED's
- 4 220-ohm resistors
- 1 button
